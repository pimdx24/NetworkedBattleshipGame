# Frontend Setup

**You must run this once before `npm run dev` will work:**

```bash
npm install
```

This downloads Vite, React, and the React plugin into `node_modules/`.
It only needs to be done once (or again after deleting `node_modules/`).

Then start the dev server:

```bash
npm run dev
```

The app runs at http://localhost:5173.
The Java server must also be running (`java BattleshipServer` in `src/`).
